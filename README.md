# Basics Java

This repository contains Java examples and exercises for learning the basics of Java programming.

## Contents

- Simple Java programs
- Core language examples
- Basic input/output
- Control flow and loops
- Object-oriented concepts

## Prerequisites

- Java Development Kit (JDK) installed
- A text editor or IDE like VS Code, IntelliJ IDEA, or Eclipse

## Setup

1. Open the project folder in your IDE.
2. Compile Java files using `javac`.
3. Run programs with `java`.

## Example

```bash
javac Main.java
java Main
```

## Notes

Keep the code organized and practice with small examples to build Java fundamentals.
